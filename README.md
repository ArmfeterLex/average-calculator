# Average Calculator

Простейший калькулятор среднего арифметического на C#. 
Запрашивает два числа, выводит их среднее с округлением до двух знаков.

- <img width="583" height="171" alt="image" src="https://github.com/user-attachments/assets/3d82799d-26e7-462d-81bd-80116a058221" />

## Возможности

- Ввод двух целых чисел с клавиатуры
- Вычисление среднего арифметического
- Округление результата до двух знаков после запятой

## Пример работы

Введите два любых числа
5
3
Среднее арифметическое этих чисел 4.00

## Использование

1. Запустить программу
2. Ввести первое число
3. Ввести второе число
4. Получить результат

## Куда можно встроить

- Как модуль в образовательный проект
- В любой калькулятор, где нужно среднее значение
- Как пример для обучения основам C#

## Требования

- .NET Core / .NET Framework
- Консоль
